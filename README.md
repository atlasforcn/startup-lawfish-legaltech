![startup-lawfish-legaltech 功能截圖](docs/preview.png)

- 這個原型在做什麼：把 Law 網之魚做成微型企業法務風險分析 SaaS。
- 特色定位：特色是把客戶案件、合約條款、法遵任務與風險分數放在同一個營運工作台。
- 操作流程：切換產業情境與客戶案件 → 檢視文件條款風險 → 標記已處理或新增法遵提醒 → 追蹤風險變化

## 核心概念

依官方名稱「法律科技分析，微型創業更安心」理解，本原型將它做成給微型企業使用的法務 SaaS：客戶案件管理、合約條款風險辨識、法遵提醒、文件版本追蹤與合規分數儀表板。

## 功能

- 依產業情境切換電商、接案工作室與餐飲品牌案例
- 客戶案件搜尋與風險排序
- 合約條款風險標記與處理狀態更新
- 法遵提醒新增與風險時間線追蹤
- 儀表板統計高風險、文件數、期限與合規分數

## 聲明
